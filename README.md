# Nextjs w/ Tailwind Template
1. `git clone ` <br>
2. `cd Nextjs-Skeleton` <br>
3. `rm -rf .git` <br>
4. `npm install` <br>
5. `npm run dev` <br>

## How I set this up 
1. `npx create-next-app@latest` <br>
2. `npm install -D tailwindcss postcss autoprefixer` <br>
3. `npx tailwindcss init -p` <br> 
4. `npm install @mantine/core` <br>
